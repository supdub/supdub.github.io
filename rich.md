## About

This page is written in **Markdown** and renders cleanly on GitHub.

You can use:

- **Bold**, *italic*, ~~strikethrough~~
- Inline code like `npm run dev`
- Blockquotes and collapsible sections
- Tables, task lists, and fenced code blocks

- Simple and readable structure
